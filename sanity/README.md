# Chronilogix Studio

Sanity Studio for managing site-wide content (currently: the hero video).

## First-time setup

```bash
cd sanity
npm install
```

## Run it

```bash
npm run dev
```

Opens at <http://localhost:3333>. Log in with your Sanity account.

## What lives in here

- **Site Settings** — singleton document with the hero video. Editing this and clicking *Publish* swaps the video on the live site (after the Next.js build picks it up).
